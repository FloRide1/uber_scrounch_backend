## About


This project is the backend of a small delivery plateform for the "Atelier Lyon".

## How to launch it

```sh
# Copy example environnement (don't forget to edit it)
cp .env.example .env

# Developement
cargo run

# Build + Production
cargo build --release
PROFILE=prod ./build/release/uber_scrounch_backend
```
